Metric Converter is a WP extension for the visual editor that allows to convert metric units to American linear measures (inch, oz, lbs).

== Description ==
Metric Converter is a great time saver for everyone who has tried AliExpress dropshipping. This extension allows you to easily convert length, weight and volume units from imperial measurement system to international one and vice versa. Now you do not have to calculate everything manually. The extension will calculate the parameters and show the result of conversion with necessary intervals, dashes and measurement units. With Metric Converter your manual product editing is greatly simplified. Convert units in a couple of clicks and enjoy perfectly optimized product information.

**Features:**
- Quick and easy unit conversion
- American and international length, weight and volume units
- Use hotkeys to convert units
- No manual calculation
- Save tons of time and get nice edited product information
- Perfect for AliExpress dropshipping stores
- Compatible with WooCommerce

== Installation ==

To install the extension, please follow this instruction.

1. Upload `metric-converter` folder to the `/wp-content/plugins/` directory, or install the extension through the WordPress plugins screen directly.
2. Activate the extension through the 'Plugins' menu in WordPress
3. The conversion options will be available in the product information

== Frequently Asked Questions ==
= What units can be converted with this extension? =
You can convert length, weight and volume units: mm, cm, m, inch, feet, cm, grams, kg, lbs, ml, l, oz.
= How to convert units with the help of Metric Converter? =
1. Select the units that you want to convert
2. Press hotkey or select necessary option from the dropdown menu
3. Enjoy converted and beautifully formatted units with intervals, dashes, and the units of measurement
= How do hotkeys work? =

1) ** cm / ** inch
Alt + 1 (mm), Alt + 2 (cm)

2) ** x ** x ** cm / ** x ** x** inch
Alt + 3

3)** x ** x ** cm / ** x ** x ** inch (bold)
Alt + Q (mm), Alt + W (cm)

4) ** (cm, regular font) / ** (inch, bold font)
Alt + A (mm), Alt + S (cm)

5) **-** (cm, regular font) / **-** 9 (inch, bold font)
Alt + Z (mm), Alt + X (cm)

6) ** kg / ** lbs
Alt + I (g), Alt + O (kg)

7) ** l / ** fl oz
Alt + K (ml), Alt + L (l)
